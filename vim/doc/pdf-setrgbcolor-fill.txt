*PDF_setrgbcolor_fill* -- Set fill rgb color values [deprecated]

  bool PDF_setrgbcolor_fill(resource p, float red, float green, float blue)~

Sets the current fill color to the supplied RGB values. Returns TRUE on
success or &false; on failure.

This function is deprecated since PDFlib version 4, use |PDF_setcolor|
instead.
