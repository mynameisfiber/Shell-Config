*SDO_Model_ReflectionDataObject::getType* -- Get the SDO_Model_Type for the
SDO_DataObject

  SDO_Model_Type SDO_Model_ReflectionDataObject::getType()~

This function is EXPERIMENTAL. The behaviour of this function, its name, and
surrounding documentation may change without notice in a future release of
PHP. This function should be used at your own risk.

Returns the SDO_Model_Type for the SDO_DataObject. The SDO_Model_Type holds
all the information about the data object's type, such as namespace URI, type
name, whether it is a primitive data type, and so on.

None.
