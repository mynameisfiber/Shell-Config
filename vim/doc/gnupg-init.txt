*gnupg_init* -- Initialize a connection

  resource gnupg_init()~

This function has no parameters.

A GnuPG resource connection used by other GnuPG functions.

Procedural |gnupg_init| example

<?php >
  $res = gnupg_init();
?>



OO gnupg initializer example

<?php >
  $gpg = new gnupg();
?>
