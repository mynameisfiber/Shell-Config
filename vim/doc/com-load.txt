*com_load* -- Creates a new reference to a COM component [deprecated]

Deprecated, use the OO syntax instead.

OO syntax

<?php >
  // do this
  $obj = new COM($module);
  // instead of this:
  $obj = com_load($module);
?>





This function does not exist in PHP 5; instead, you should use the regular and
more natural OO syntax to access properties or call methods.
