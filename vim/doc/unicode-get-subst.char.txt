*unicode_get_subst_char* -- Get the substitution character for string conversion
errors

  unicode unicode_get_subst_char()~

Return the substitution character for string conversion errors set by
|unicode_set_subst_char|.

The unicode substitution character is returned.

This function is EXPERIMENTAL. The behaviour of this function, its name, and
surrounding documentation may change without notice in a future release of
PHP. This function should be used at your own risk.
