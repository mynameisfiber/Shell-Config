*xslt_backend_info* -- Returns the information on the compilation settings of
the backend

  string xslt_backend_info()~

|xslt_backend_info| gets information about the compilation settings of the
backend.

Returns a string with information about the compilation setting of the backend
or an error string when no information available.

|xslt_backend_name| |xslt_backend_version|
